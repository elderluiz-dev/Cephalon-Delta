Need help with Cephalon Delta?

Cephalon Delta is an open-source companion application for Warframe players who want a simple way to track Prime farming progress. It helps you organize your active Prime sets, monitor collected parts, and keep track of their Ducat values for future Baro Ki'Teer visits.


This project was built as both a learning experience and a tool for the community. While it currently focuses on collection tracking, new features may be added over time based on community feedback.


If you encounter a bug, have a suggestion, or would like to contribute, feel free to visit the project's GitHub repository or join the discussion with the community.
